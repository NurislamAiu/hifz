Если у вас возникла проблема с приложением Hifz, напишите через GitHub Issues ниже.

Contact

Support: github.com/NurislamAiu/hifz/issues

Before contacting support

  • Қолданбаның соңғы нұсқасын орнатыңыз.
  • Интернет байланысын тексеріңіз, әсіресе аудио жүктеу кезінде.
  • Мәселені түсіндіріп, құрылғы моделін және iOS нұсқасын жазыңыз.

Перед обращением

  • Установите последнюю версию приложения.
  • Проверьте интернет-соединение, особенно при загрузке аудио.
  • Опишите проблему, модель устройства и версию iOS.
